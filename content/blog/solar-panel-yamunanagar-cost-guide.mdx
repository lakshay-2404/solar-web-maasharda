## System type ke hisaab se cost

| System Type | Practical Budget | Note |
| --- | --- | --- |
| On-grid | ~Rs 45,000/kW se shuru | Basic market reference |
| DCR on-grid | ~Rs 58,000-Rs 60,000/kW | Subsidy-compliant residential planning |
| Off-grid | ~Rs 60,000-Rs 70,000/kW | Battery cost add hoti hai |
| Hybrid | ~Rs 65,000-Rs 80,000/kW | Grid + backup dono |

Off-grid aur hybrid systems mehange isliye hote hain kyunki battery, inverter architecture aur backup planning alag hoti hai. On-grid sabse simple hota hai, isliye uski entry cost relatively low rehti hai.

Lekin agar target subsidy lena hai, to DCR compliance ignore nahi ki ja sakti. Isi point par normal quote aur workable quote alag dikhne lagte hain.

## 3kW DCR system ka real example

3kW size sabse popular category hai, lekin yahan ek common misunderstanding hoti hai. Kai log old generic numbers dekhkar maante hain ki subsidy ke baad cost bahut niche chali jaayegi, jabki compliant DCR setup ka real budget usse upar hota hai.

## DCR vs non-DCR difference

Market mein kuch non-DCR quotes 10%-15% cheaper lag sakte hain. Pehli nazar mein woh attractive dikhte hain, lekin subsidy-linked project mein non-DCR lena seedha नुकसान ban sakta hai.

Seedha rule yaad rakho: **non-DCR lagaya to subsidy nahi milegi**. Aur agar subsidy miss hui to Rs 78,000 tak ka direct benefit chala jaata hai.

Isliye sirf quote compare karna kaafi nahi hota. Quote ke saath compliance bhi compare karna zaruri hota hai.
